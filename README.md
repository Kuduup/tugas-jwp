2. Dashboard / Home
Setelah login, muncul halaman utama dengan pesan sambutan: "Selamat datang, admin".

Sidebar menu berisi pilihan:

- Home

- Daftar Barang

- Tambah Barang

- Logout

Instruksi: pilih menu di sidebar untuk mulai mengelola data barang.

3. Daftar Barang (Kelola Barang)
Menampilkan tabel berisi daftar barang dengan kolom:

- Nomor

- Nama

- Kategori

- Jumlah

- Harga

- Aksi (Edit / Hapus)

Tombol + Tambah Barang tersedia di atas tabel untuk menambahkan barang baru.

Tombol Kembali untuk kembali ke dashboard.

4. Tambah Barang
Form input dengan field:

- Nama

- Kategori

- Jumlah

- Harga

Tombol Simpan â†’ menyimpan data barang baru ke sistem.

Tombol Kembali â†’ kembali ke daftar barang.

5. Edit Barang
Form edit menampilkan data barang yang sudah ada (misalnya: Nama = minyak, Kategori = sembako, Jumlah = 10, Harga = 11000).

Admin bisa mengubah detail barang sesuai kebutuhan.

Tombol Update â†’ memperbarui data barang di sistem.

Tombol Kembali â†’ kembali ke daftar barang.

6. Hapus Barang
Di tabel daftar barang, setiap baris memiliki tombol Hapus.

Jika diklik, barang tersebut akan dihapus dari sistem.

Setelah dihapus, tabel akan diperbarui otomatis.

7. Logout
Menu Logout di sidebar digunakan untuk keluar dari sistem.

Setelah logout, admin kembali ke halaman login.
